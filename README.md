### Development

Developed with the following software and versions:

- live-server 0.9.0
- node v5.11.1
- npm 3.8.6
- typings v1.3.0

Get started:

``` bash
# install
$ git clone https://github.com/deificx/sprite
$ cd sprite
$ typings install
$ npm install
# npm start will start a watching webpack process
$ npm start
# optionally run live-server in a new terminal window
$ live-server
```
